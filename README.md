# life-blueprint-destiny

一个 Codex Skill：先基于出生信息做正统命盘解读，再转译为人生设计蓝图。

它适合这些场景：

- 用户输入出生年月日时、出生地，希望结合命盘看人生方向。
- 用户想把八字、紫微斗数、西洋占星等命理信息转成可行动的人生蓝图。
- 用户关心职业方向、长期定位、人生阶段选择、三条五年人生版本。

## 核心原则

这个 skill 的定位是：

> 命盘为底，人生设计为用。

它不会把命盘当成宿命判决，而是把命盘作为结构化自我观察工具。报告必须落到人生设计：真问题、指南针、能量地图、奥德赛计划和原型行动。

## 支持体系

- 八字四柱：默认体系
- 紫微斗数：需要可靠排盘结果或用户提供命盘
- 西洋占星：需要可靠 natal chart 或用户提供星盘数据
- 生命灵数：只作为轻量补充，不作为主依据

## 安装

把整个文件夹复制到 Codex skills 目录：

```bash
cp -R life-blueprint-destiny ~/.codex/skills/
```

或者把本仓库克隆到 skills 目录后重命名：

```bash
git clone <repo-url> ~/.codex/skills/life-blueprint-destiny
```

## 使用示例

```text
用 life-blueprint-destiny，根据我的出生信息生成一份人生蓝图。
我的出生信息：1993年8月12日上午9:00，出生在浙江省杭州市。
性别：可选
最想解决的问题：职业方向与长期定位
健康：7
工作：5
娱乐：8
爱：7
```

## 输出内容

完整报告通常包括：

- 排盘依据与可信度
- 正统命盘解读
- 命理作为镜子的待验证假设
- 四个仪表盘：健康、工作、娱乐、爱
- 真问题：重力现实 vs 可设计问题
- 工作观、人生观与正北方向
- 能量地图
- 三个五年奥德赛计划
- 优先原型与行动清单
- 失败免疫提醒

## 边界

- 不做确定性预测。
- 不编造四柱、十神、大运、宫位、星曜、行星宫位。
- 没有可靠排盘时，会要求用户提供命盘截图或文本。
- 不替代医疗、法律、财务等专业建议。
